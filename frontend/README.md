# KHILADI Frontend

KHILADI is a professional Taekwondo Tournament Manager frontend built with React and Vite.

## Features

- User login/register
- Google login
- Tournament creation
- Entry management
- Tie-sheet generation
- Winner/result pages
- Team championship calculation
- Team submission approval/rejection
- Admin dashboard
- PDF/print support

## Tech Stack

- React
- Vite
- Axios
- React Router
- TanStack Table
- CSS Modules

## Setup

```bash
npm install
npm run dev
